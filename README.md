# AIDA - Site web officiel

Site web statique pour l’organisation AIDA (Aide Initiative des Développements Autonomes).

## Contenu
- `index.html` : page principale
- `styles.css` : styles et mise en page responsive
- `script.js` : menu mobile et validation simple du formulaire

## Hébergement recommandé
Le site peut être hébergé facilement en mode statique sur :
- Render (Static Site)
- Netlify
- GitHub Pages

## Déploiement rapide sur Render
1. Créer un dépôt Git avec ces fichiers.
2. Créer un nouveau site statique sur Render.
3. Connecter le dépôt Git.
4. Déployer.

## Notes
- Le formulaire de contact est en front-end. Pour un vrai envoi d’email, il faut connecter un service d’e-mail ou utiliser un backend.
- Ce site est conçu pour être simple et adapté aux hébergements gratuits.
